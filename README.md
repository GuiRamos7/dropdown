##Access to the project

To run the project follow the nexts steps

#### Cloning the project

```bash
    $ git clone https://github.com/GuiRamos7/dropdown.git
```

#### Go to the project

```bash
    $ cd dropdown
```

#### Open your IDE (My case is VSCode)

```bash
    $ code .
```

### Run the project

```bash
    $ yarn && yarn start
```

### Run the server
```bash
    $ yarn && yarn server
```
